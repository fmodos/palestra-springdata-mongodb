palestra-springdata-mongodb
====================

Palestra Spring Data com MongoDB

Detalhes dos arquivos
--------------
- SpringData MongoDB.pdf: slides da apresentação
- dishMongoDB: folder com o projeto java que contém os códigos fontes utilizados na apresentação

É preciso ter o mongodb instalado, link com o instalador: http://www.mongodb.org/downloads.

O projeto dishMongoDB está configurado para o eclipse, basta somente importar ele e executar as classes do folder test.
